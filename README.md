# 🚀 CryptoDash | Live Portfolio Tracker

Интерактивный дашборд для отслеживания криптовалютного портфеля в реальном времени. Проект создан на **Vue 3** с использованием **Vite** и интеграцией **CoinGecko API**.

## ✨ Основные фичи
* **Live Prices:** Автоматическое обновление курсов валют каждую минуту через CoinGecko.
* **Dynamic Allocation:** Визуализация распределения активов с помощью круговой диаграммы (Chart.js).
* **Persistence:** Твой портфель сохраняется в `localStorage` — данные не пропадут после перезагрузки страницы.
* **Customization:** Возможность добавлять любые монеты по их API ID и настраивать цвета графиков.
* **Responsive Design:** Полная адаптивность под мобильные устройства и десктопы.

## 🛠 Стек технологий
* **Framework:** Vue 3 (Composition API, `<script setup>`)
* **Bundler:** Vite
* **Charts:** Chart.js + Vue-Chartjs
* **Styles:** Scoped CSS (Dark Theme, Glassmorphism elements)
* **Deployment:** GitHub Pages
